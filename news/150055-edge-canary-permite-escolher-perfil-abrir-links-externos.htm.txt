A Microsoft lançou uma atualização para o Edge Canary, versão beta com atualizações diárias do navegador, que passa a permitir preferências diferentes em cada perfil. O Edge Canary já permite a criação de vários perfis para fins específicos como, por exemplo, para uso pessoal ou profissional . O último update permite selecionar um perfil padrão para abrir links externos.
Para definir um perfil padrão, o usuário deve ir em Settings (Configurações), seguido de Profiles (Perfis), depois em Multiple profile preferences (Preferências de Perfil Múltiplo) e, por fim, selecionar o “default profile for external links” (perfil padrão para links externos).
A empresa também adicionou a seção "Phones and other devices” (Telefone e outros dispositivos) em suas configurações. A opção permite habilitar a sincronização do navegador nos dispositivos Android, iOS e computadores com Windows e MacOS para acessar, os favoritos do navegador, senhas, endereço, entre outras funções. A seção também oferece um QR Code que redirecionará os usuários para o aplicativo Edge na loja de aplicativos.
Além da versão Canary, versão beta, o navegador também está disponível nas versões Canal Dev, com atualizações semanais, o que torna o navegador mais estável. A empresa também oferece o navegador na versão Canal Beta, a mais estável entre as prévias do Edge, que conta com atualização realizada a cada 6 semanas, e reúne as melhorias consolidadas das compilações Dev e Canary.
O desenvolvimento da programação do novo Edge, a partir de janeiro de 2020, é baseado no Chromium, projeto de código aberto do Google que deu origem ao navegador Chrome. Com isto, muitas das funcionalidades do Chrome já podem ser vistas no produto da Microsoft, e algumas funções ainda estão sendo desenvolvidas.
O navegador está disponível nos sistemas operacionais móveis Android, iOS e de computadores Windows 10, 8 / 8.1, 7, e Windows Server a partir de 2008, além de MacOS.
